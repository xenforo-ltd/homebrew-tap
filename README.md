# XenForo Tap

## How do I install these formulae?

`brew install xenforo-ltd/tap/<formula>`

Or `brew tap xenforo-ltd/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "xenforo-ltd/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
